## Introduction à JavaScript

### Qu’est-ce que JavaScript ?

Aujourd’hui, JavaScript est devenu un langage incontournable du développement moderne. Il est utilisé :

* côté **client** (navigateurs web),
* côté **serveur** grâce à **Node.js**,
* pour des applications web, mobiles et même desktop.

### Caractéristiques principales

JavaScript est :

* **Interprété** : il n’a pas besoin d’être compilé avant l’exécution,
* **Typé dynamiquement** : une variable peut changer de type au cours du programme,
* **Faiblement typé** : les conversions de types sont souvent implicites,
* **Multi‑paradigme** : il supporte la programmation procédurale, orientée objet et fonctionnelle.
